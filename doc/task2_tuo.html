Task2 - 障碍物避免与探索系统

文件: scripts/task2.py | ROS2 节点: obstacle_avoidance | 状态: 开发中

概述

基于 Braitenberg Vehicle 3b 策略的 TurtleBot3 自主探索系统，在 4x4m 竞技场中实现 90 秒内的障碍物避免和区域探索。

功能 状态 描述
LiDAR 数据处理 已完成 7 个方向距离测量（前、后、左、右、前左、前右、后）
里程计跟踪 已完成 实时位置 (x, y) 和偏航角 (yaw) 跟踪
基础移动函数 已完成 时间/距离/角度控制的移动和旋转
障碍物检测 已完成 线程安全的 LiDAR 数据访问
探索策略 进行中 当前为测试序列，需要实现完整探索算法
区域跟踪 待完成 12 个 1x1m 探索区域的访问跟踪

核心功能模块

1. LiDAR 数据处理

# 270 度扫描，索引映射 front_idx = 0 # 0° - 正前方 left_idx = 90 # 90° - 左侧 right_idx = 270 # 270° - 右侧 back_idx = 180 # 180° - 后方 front_left_idx = 315 # -45° front_right_idx = 45 # +45°

使用 lidar_lock 确保线程安全访问 LiDAR 数据。

2. 里程计跟踪

# 从四元数转换为欧拉角 _, _, self.yaw = quaternion_to_euler(orientation) self.x = position.x self.y = position.y

使用 odom_lock 确保线程安全访问位置数据。

3. 移动控制函数

函数 功能 参数 障碍物检测
rotate_by() 按时间旋转 direction, ang_vel, duration 无
go_by() 按时间前进/后退 direction, lin_vel, duration 有（前方）
go_until() 前进指定距离 distance, err 有（前方）
roatae_unitl() 旋转指定角度 direction, deg, err 无
